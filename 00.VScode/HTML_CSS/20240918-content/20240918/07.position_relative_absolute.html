position:relative(相對定位)

使用位移屬性(top、right、bottom、left) 會以自己原來的位置位移，且保留自身的空間進入定位層，原本位置也不會被別人佔據。

父元素沒有任何位移屬性，會留在原地不動，這種 reative 的做常作為 adsolute 子元素位移目標的依據。

進入定位層可以使用 z-index(元素可以遮住元素)

position:absolute(絕對定位)

沒有任何移位屬性(top、right、bottom、left)，該元素會脫離正常流向，提升為定位層，後方元素會依據 css 流動的特性流動至絕對定位元素的下方(背絕對定位)元素遮住

若祖父元素找不到 absolute、relative、fixed，最後會以初始容器根元素(html)作為位移目標

父元素會用 relative 與 子元素 absolute 搭配的原因是，父元素可以維持在原來的位置，而 absolute、fixed 會脫離正常流向。
