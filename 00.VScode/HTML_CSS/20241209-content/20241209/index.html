01.Sass 與 SCSS

1.兩個都稱為 Sass，是 SCSS 的前身。

2.瀏覽器看不懂 Sass、SCSS，所以要經過編譯成 css。

3.Sass 是御前處理器，目的在減少 CSS 的重工，以及更有效率維護樣式的結構。

4.可以使用變數、函式、流程控制、巢狀(嵌套)、混入(mixins)...。

02.資料型態

  1. 1.number(數字):有分有單位、無單位，1、12、10px、20px
  2. 2.string(字串):有分有引號、無引號，"Noto Sans TC"、Noto Sans TC
  3. 3.boolean(布林值):true、false
  4. 4.null(空值):null
  5. 5.color(顏色):#fff、red、rgb()、rgba()
  6. 6.list(列表，像陣列):有分半形或者逗號，1.5rem 1rem 0 2rem，'Couruer New'，Couruer，monospace。
  7. 7.map(物件，像js的物件)

03.變數

宣告用 $，宣告位置就是它的作用域，寫在最外面式全域，大括號裡面是區域。

變數的底線 _ 和 連接線 -，可以互換使用，$bg_dark、$bg-dark 兩者是一樣的變數。

使用變數可以維持網站風格的一致性，節省修改成本，只需修改變數，不需要再 CSS 修改。

04.流程控制

@if、@else if、@else

判斷條件可以括號、可以不括號

使用 and、or、not 處理判斷運算式的結果是 true 或 false

box

@for 迴圈有兩種 through 與 to

會包含 end number

不包含 end number

06.@extend 繼承 %

繼承可以用 CSS 選擇器或 % 佔位選擇器來命名

使用 CSS 選擇器會被編譯到 CSS，佔位符屬於 SCSS 的範疇，不會被編譯出去。

元素有共同樣式可以使用繼承提取出去，會被匯聚到群組選擇器。

07.計算

可以加、減、乘、除、取餘數的計算

08.函式 @function

@function 與 @mixin 兩者使用方式很像，但函式必須要使用 @return 返回結果，@mixin 不需要返回結果。

@function 用於計算結果，@mixin 用於產生一段 CSS 樣式。
