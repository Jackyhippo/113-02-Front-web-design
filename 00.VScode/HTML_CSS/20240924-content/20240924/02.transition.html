轉場(transition)或稱為過渡

  • 轉場是兩組樣式之間漸進式的轉換過程，起始樣式、結束樣式，瀏覽器會將這兩組樣式形成捕間動畫(Tween)。
  • 必須要有狀態觸發動作(狀態有很多種)，最常見 :hover，或是 media query (rwd) 媒體查詢尺寸也可以造成轉場。
