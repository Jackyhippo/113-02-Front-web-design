背景圖 background-size

調整背景圖與容器一樣大小時，可以設定 background-size: cover。

圖片 object-fit

1.設聽圖片 width、height 100% 占滿容器，但可能圖片與容器比例不對，會造成圖片變形。

2.對圖片使用 object-fit:cover，再使用 object-position 調整位置。
