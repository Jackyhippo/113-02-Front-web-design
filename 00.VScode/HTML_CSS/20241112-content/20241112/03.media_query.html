響應式網頁設計 (Responsive web design, RWD)

  • 1.響應式網頁設計是讓內容如水般能適應各種形狀的容器，在每個裝置尺寸上能正常瀏覽。
  • 2.媒體查詢(media query)負責響應式主要工作
  • 3.設定斷點(breakpoint)為不同裝置尺寸設定通過的位置點，當通過該位置套用該尺寸的樣式。
  • 4.使用單位設定寬高或大小(rem、%、vh、vw)
